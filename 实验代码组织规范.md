```python
    def __init__(self, dice_weight=0.5, focal_weight=0.5, lovasz_weight=0.5):
        super().__init__()
        self.dice_loss = DiceLoss()
        self.focal_loss = FocalLoss()
```

## 5. 评估指标 (`metrics.py`)

### 职责
计算分割任务的评估指标。

### 规范
- 输入为模型输出（logits 或概率）和标签
- 返回 IoU、Precision、Recall、F1 等指标

### 示例
```python
def calculate_metrics(pred_img, mask, threshold=0.5):
    pred_binary = (pred_img > threshold).float()
    mask = mask.float()

    pred_flat = pred_binary.view(-1)
    mask_flat = mask.view(-1)

    tp = (pred_flat * mask_flat).sum()
    fp = (pred_flat * (1 - mask_flat)).sum()
    fn = ((1 - pred_flat) * mask_flat).sum()

    epsilon = 1e-6
    iou = (tp + epsilon) / (tp + fp + fn + epsilon)
    precision = (tp + epsilon) / (tp + fp + epsilon)
    recall = (tp + epsilon) / (tp + fn + epsilon)
    f1 = 2 * precision * recall / (precision + recall + epsilon)

    return iou.item(), precision.item(), recall.item(), f1.item()
```

```python
def train_one_epoch(model, train_loader, optimizer, loss_fc, device, epoch):
    model.train()
    total_loss = 0.0
    iou, precision, recall, f1 = 0.0, 0.0, 0.0, 0.0
    monitor_stats = {}

    pbar = tqdm(train_loader, desc=f"Training Epoch {epoch}")
    for batch_idx, batch in enumerate(pbar):
        opt_imgs, sar_imgs, labels, _ = batch

        # 数据移至设备
        opt_imgs = opt_imgs.to(device)
        sar_imgs = sar_imgs.to(device)
        labels = labels.to(device)

        # 前向传播
        optimizer.zero_grad()
        outputs, stats, _ = model(opt_imgs, sar_imgs, return_stats=True)

        # 计算损失
        loss, loss_dice, loss_focal, loss_lovasz = loss_fc(outputs, labels)

        # 反向传播
        loss.backward()
        optimizer.step()

        # 累计指标
        total_loss += loss.item()
        cur_iou, cur_precision, cur_recall, cur_f1 = calculate_metrics(outputs, labels)
        iou += cur_iou
        # ...

    # 返回平均值
    num_batches = len(train_loader)
    return total_loss / num_batches, iou / num_batches, ..., monitor_stats
```

## 关键设计原则

### 1. 配置与代码分离
所有可调参数集中在 `config.py`，便于实验管理。

### 2. 可复现性保证
- 固定随机种子
- 训练集统计信息缓存
- 验证/测试集复用训练集统计

### 3. 实验监控
- W&B 在线看板
- 飞书实时通知
- 本地日志文件

### 4. 模块化设计
- 每个文件职责单一
- 组件可复用（如损失函数、指标计算）

### 5. 训练监控统计
模型 forward 支持 `return_stats` 返回内部激活、梯度分布等信息：
```python
logits, stats, viz_data = model(opt, sar, return_stats=True)
# stats 包含: attention entropy, gate weights, fusion weights 等
```

### 6. 大图推理策略
- 滑动窗口切片
- 高斯权重融合（减少边缘伪影）
- 批量处理提高效率
